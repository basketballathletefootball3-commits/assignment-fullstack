# Frontend

React + Vite frontend for the assignment demo.

## Quick Start

```bash
cd frontend
npm install --legacy-peer-deps  # because of vite version mismatch
npm run dev
```

The app will be at `http://localhost:5173`.

## Features

- User registration and login with JWT
- Task CRUD (Create, Read, Update, Delete)
- Real-time task list updates
- Basic styling with inline styles

## Build

```bash
npm run build
npm run preview
```
